关于应用 v2rayW

简介

本项目为 ProjectV(迁移到 V2fly) 项目中 v2ray-core 的 web 客户端。通过网页端配置代理协议进行科学上网。 目前支持 vmess,vless 协议，其他协议和更多功能将之后进行迭代开发。

使用

Step1: 设置浏览器的代理协议，默认代理协议为 socks5 暂不支持修改。 默认代理端口为 1080 ，可以通过高级设置进行修改！

Step2: 在服务管理里配置好正确的代理协议，点击启动按钮。

其他

当前 v2rayW 版本为 v1.3.1 版本更新详情说明见 release。 v2ray-core 版本为 v4.29.0

若在使用过程中发现任何 bug ，或需要 v2rayW 支持的功能欢迎你向 github 提交 issue 进行说明。
